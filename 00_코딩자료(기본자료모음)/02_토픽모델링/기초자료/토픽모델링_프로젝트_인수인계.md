## 💡 구현 방향

**역할 분담:**
- **Claude Code** → 대량 기사 수집 + 전처리 + 저장
- **Claude API** → 수집된 기사 토픽 추출 + 트렌드 분석

**흐름:**
```
Claude Code로 수집 (기간: 최근 1~2년치)
    ↓
전처리 (중복 제거, HTML 태그 제거)
    ↓
Claude API로 토픽 분석 요청
    ↓
결과 리포트 출력
```

## 🔧 Claude API 준비사항

- `console.anthropic.com` 에서 API 키 발급
- 크레딧 충전 ($5 이상)
- `.env`에 `ANTHROPIC_API_KEY=sk-ant-xxxxx` 추가
- `pip install anthropic`

**비용:** 기사 200개 분석 기준 $0.01 미만 (매우 저렴)
**추천 모델:** claude-haiku-4-5 (저렴) 또는 claude-sonnet-4-20250514 (고품질)
