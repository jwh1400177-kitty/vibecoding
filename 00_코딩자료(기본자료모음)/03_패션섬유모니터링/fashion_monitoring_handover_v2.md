## 🔄 파이프라인 구조 (8단계)

```
1단계: 수집
       네이버 API (키워드별 display 차등)
       Google RSS (무제한)
       다음 4단계 폴백
       전문지 17개 RSS
       ↓
2단계: HTML 태그 정리
       ↓
3단계: 날짜 필터 (target_dates 범위만 통과)
       ↓
3-2단계: 블랙리스트 필터 (전문지 제외)
       ↓
4단계: 키워드 검증 필터
       네이버/구글: 전량 통과
       다음: title OR summary에 키워드 있으면 통과
       전문지: title에 키워드 있으면 통과
       ↓
4-2단계: 화이트리스트 필터
       전문지: 무조건 통과
       화이트리스트 면제 브랜드 9개: 무조건 통과
       나머지: 화이트리스트 적용
       ↓
5단계: URL 중복제거 → 각 탭 저장
       ↓
6단계: 5개 탭 합산 → 교차 중복제거 → 전체종합
       ↓
6-2단계: 최종전송 200건 선별 → 카톡전송 → 이메일 발송
```
